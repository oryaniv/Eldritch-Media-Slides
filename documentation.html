listed here is all you need to know about the methods and properties supplied to you by Eldritch media slides.

Module EldritchSlideShow

EldritchSlideShow is the global object suppldied to you by the library. This object is used primarily for the creation of new slidshows, and also for configuartion of global properties for all slideshows.

methods

Create()

Signature Create(queryString : string (optional), sd : SlideShow)
Description creates a brand new slideshow, and attaches it to the target element, or appends it to the body if no queryString was supplied.
Return Value SlideShowRemote object
Parameters
  • queryString(optional) : a jquery/css string indicating an html element to contain the slideshow
  • SlideShow : an object containing all slideshow properties
Example 
                

                   
 

                   EldritchSlideShow.Create("#container",{
                        Images:["img1.jpg","img2.jpg","img3.jpg","img4.jpg","img5.jpg"],
                        Size:4,
                        Animations:3,
                        Filters:2,
                        Controls:1
                   })

Config()

Signature Config(property : string,value : string)
Description changes the path/url used by Eldritch media slides to find script and style files
Return Value void
Parameters
  • property : "stylePath" for changing css lookup and "scriptPath" for changing jd lookup
  • value : the new path/url should be used to get the files/li>
Example 
                

                    EldritchSlideShow.Config("stylePath","myServerPath/styles");                 
                    EldritchSlideShow.Config("scriptPath","myServerPath/scripts");

Class SlideShow

objects of this class are the material used to create new slideshows, and are given as parameters to the create() method.

fields

name : type description default value
Images: Array<string>
an array of paths/urls for images
 []
Slides: Array<Slide>
an Array of objects of Slide Class
 []
SlideTexts : Array<SlideTitles>
an Array of objects of Slide Title Class
 []
Backgrounds : Array<BackGround>
an Array of objects of Background Class
 []
Animations: number a number describing the level of slide animations allowed
  • 0 : None
  • 1 : basic
  • 2 : standard
  • 3 : all
 0
Filters: number a number describing the level of filters/middle animations allowed
  • 0 : None
  • 1 : basic
  • 2 : standard
  • 3 : all
 0
TextStyles: number a number describing which pre-defined text styls are allowed
  • 0 : Normal
  • 1 : basic
  • 2 : standard
  • 3 : flamboyant
 0
TextAnimes: number a number describing which text animations are allowed
  • 0 : None
  • 1 : basic
  • 2 : all
 0
Size: number a number defining the a fixed size for the slideshow
  • 0 : not specified*
  • 1 : small
  • 2 : medium
  • 3 : large
  • 4 : Fullscreen
 0
Audio : string
a path/url for an audio track for the slideshow
 ""
Loop : boolean determines if slideshow will play in loops false
Controls : number determines if control buttons will appear on the slideshow for the user to control it
  • 0 : not visible
  • 1 : visible only when mouse over
  • 2 : always visible
 0
backgroundbetween : boolean determines if a background will appear between slide transitions false
CssClass : string
a Css class to be attached to the slideshow element. this can override the size property defined above
 ""
CallBack : object an object including a function name and an array of parameters. the function will be called with the supported parameters when the slideshow ends {}
Poster : string
a url/path to an image to appear as a placeholder for the slideshow when in does not play
 ""

Class Slide

objects of this class are the composition of an image, and some additional properties that lets you decide how to show it as a slide

fields

name : type description default value
path : string
a path/url to an image resource
 ""
text : string text to appear on the slide. At default it will appear on the center of the lower half of the screen. ""
lifetime : number time (in seconds) until the slide will start to leave the screen 8

Class SlideTitle

Objects of this class represent custom texts that will appear on the slideshow with no regard to the slides. This allows you to add your custom comments to the show where ever you want them,

fields

name : type description default value
text : string
the text to appear
 ""
lifetime : number time (in seconds) until the text will disappear 8
style : string a css string, formatted as attribute:value;attribute:value that will be attached to the text element. this allow you to fully customize the text labels ""

Class SlideShowRemote

objects of this class are the most important of all, they are created by the create() method mentioned before, and contain both the SlideShow object, and also a rich API that allows you to control and update the slideshow. like a remote control.

methods

Start()

Signature Start()
Description starts the slideshow, from first slide on until it ends
Return Value this method returns the remote object itself, so method chaining is possible
Parameters none
Example 
                    
                        var remote = EldritchSlideShow.Create({Images:images});
                        remote.Start();

Stop()

Signature Stop()
Description stops the slideshow, resetting all progress and resuming to first slide
Return Value this method returns the remote object itself, so method chaining is possible
Parameters none
Example 
                    
                        var remote = EldritchSlideShow.Create({Images:images}); 
                        remote.Start(); 
                        setTimeout(function(){remote.Stop();},10000);

Replay()

Signature Replay()
Description stops the slideshow, then immediately starting it again
Return Value this method returns the remote object itself, so method chaining is possible
Parameters none
Example 
                
                    var remote = EldritchSlideShow.Create({Images:images}); 
                        remote.Start(); 
                        setTimeout(function(){remote.Replay();},10000);

AddImages()

Signature AddImages(images : Array)
Description adds images to the slideshow. The additional images will appear next time the slideshow runs
Return Value this method returns the remote object itself, so method chaining is possible
Parameters an array of image urls/paths
Example 
                    
                        var remote = EldritchSlideShow.Create({Images:images}); 
                        remote.Start(); 
                            setTimeout(function(){
                            remote.AddImages(["additionalImage1.jpg","additionalImage2.jpg"]);
                            remote.Replay();
                        },10000);

AddSlides()

Signature AddSlides(slides : Array )
Description adds slides to the slideshow. The additional slides will appear in the next run
Return Value this method returns the remote object itself, so method chaining is possible
Parameters an array of Slide class objects
Example 
                
                    var remote = EldritchSlideShow.Create({Slides:slides}); 
                    remote.Start(); 

                    setTimeout(function(){
                        remote.AddSlides([
                            {path:"additionalslide.jpg",text:"additinal slide 1"}, 
                            {path:"additionalSlide1.jpg",text:"additionl slide 2"}
                        ]); 
                        remote.Replay();
                    },10000);

AddSlideTexts()

Signature AddSlideTexts(slideTexts:Array)
Description adds more custom texts to the slideshow. The new texts will appear in the next run
Return Value this method returns the remote object itself, so method chaining is possible
Parameters an array of SlideTitle class objects
Example 
                    
                        var remote = EldritchSlideShow.Create({Images:images}); 
                        remote.Start(); 

                        setTimeout(function(){
                            remote.AddSlideTexts([

                                                        text:"Additional Text2",
                                                        style:"color:red,font-size:16px;"+
                                                        "margin-left:auto;margin-right:auto"
                                                    }
                                                 ]); 
                        remote.Replay();
                        },10000);

AddBackGrounds()

Signature AddBackGrounds(BackGrounds:Array)
Description adds additional backgrounds to the slideshow. The additional backgrounds will only appear in the next run
Return Value this method returns the remote object itself, so method chaining is possible

SetCallBack()

Signature SetCallBack()
Description sets the callback object, which comprises of a function name and an array of parameters
Return Value this method returns the remote object itself, so method chaining is possible
Parameters
    an object that contains 2 keys: Function which is a function name, and Params which is an array of parameters
Example 
                    
                        function foo(a,b,c){
                        // do something with a,b and c...
                        } 
                        var remote = EldritchSlideShow.Create({Images:images}); 
                        remote.Start(); 
                        setTimeout(function(){remote.SetCallBack({
                                Function:"foo",Params:[5,"boom"],20});
                            },10000);
