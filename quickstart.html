Installation

getting started with Eldritch Media Slides is easy as pie. currently there are two alternatives:

using CDN

In order to use Eldritch Media Slides while serving the script and style files straight from the cloud, just include the following in your page:

    
        <script src="https://gitcdn.link/repo/oryaniv/Eldritch-Media-Slides/master/EldritchMediaSlides.js">
        </script>

and that's it, you have all you need, just use the API and have fun!

using local files

Eldritch Media Slides can also be served directly from your file system. in order to do this, first download the required files here:

include EldritchMediaSlides.js in your page, and then include the following in your script: 

    
        EldritchSlideShow.Config("stylePath","myServerPath/styles"); 

        EldritchSlideShow.Config("scriptPath","myServerPath/scripts");

while "myServerPath/styles" is assumed to be the path where you will serve the style files, and should also contain the css file you downloadad, and "myServerPath/scripts" is assumed to be the path where you will serve the script files, and should also contain the javascript file you downloadad.

the purpose of this is to make sure that when a slideshow is created, it will look for the files it needs just where you want it to be served from.

creating your first Eldritch slide show

the most basic

well now, open up a fresh html, include the source files (however you chose to) and write the following js code:

    
        EldritchSlideShow.Create(
            {Images:
                    [
                        "img1.jpg",
                        "img2.jpg",
                        "img3.jpg",
                        "img4.jpg",
                        "img5.jpg"
                    ]

            }).Start();

assuming that the strings in the code above are paths or urls to your images. now run the code, aaand...
