version details

currently, Eldrtich media slides is only beta.

and yet, it already has all the following features:

huge amount of unique animations
slide texts
animated slide texts
styled texts
background audio
background images
button controller
easy to use API
custom size
image filtering
easy embedding

hopefully, this project will flourish with many users finding interest in the things this library can do for them. when it does, I can go on to implement some seriously amazing stuff, like:

video slides
audio playlists
custom configuration per slide
performance optimization for greater loads of slides
pause button (not just stop and play)
in-play slide navigation
slide transition by touch

I also plan for a few related projects which may be worth of the time of the above goes smoothly:

a GUI desktop based/web application for easy slide creation
a slide to video processor, which will let you to convert your slideshows into mp4
